## 七、策略联动规则（硬编码）

### 可同时运行
| 组合 | 规则 |
|------|------|
| 趋势三剑客 | 方向一致共振，总杠杆≤3x。双信号各0.5%，三信号各0.33% |
| 高波动双刃 | 同上 |
| 基座×任何 | 费率策略独立开仓，总杠杆≤3x |

### 冲突隔离
| 冲突类型 | 触发 | 动作 |
|---------|------|------|
| 顺势vs反转 | 趋势衰竭反转出信号 | 立即平顺势仓→开反向仓 |
| 趋势vs震荡 | AI判断状态切换 | 平旧→开新，冷却期12h |
| 高波动vs震荡 | AI判断状态切换 | 同上 |

### 冲突处理优先级
1. 最高：趋势衰竭反转出信号 → 强制平仓顺势
2. 高：AI判断状态切换 → 平旧开新，12小时冷却
3. 中：同状态内共振 → 降仓位
4. 低：基座策略出信号 → 独立开仓

关键坑点摘要：
1. OKX 必须走代理，直连不可靠
2. CCXT 4.x + aiohttp 会读 Windows 注册表代理 → 导致错误代理配置

5. save_config 必须合并模式，不能覆盖
6. JSON 序列化 NaN/Inf 需 safedump 处理
7. 5次连续无提升→自动停止优化
